Theorem 16.1.1 – Triangles on the Same Base and Between the Same Parallels Have Equal Areas

Book Statement

Triangles on the same base and between the same parallels are equal in area.

Key Vocabulary

  • Same base: Two triangles share a common side as their base.
  • Between the same parallels: The base lies on one line and the other vertices lie on another line parallel to the base.
  • Altitude (height): The perpendicular distance from a vertex to the line containing the base.
  • Area of a triangle: Given by \(\tfrac{1}{2} \times \text{base} \times \text{height}\).

If triangles ABC and ADC share base AC and the points B and D lie on a line parallel to AC, then [△ABC] = [△ADC].

What We Will Prove

  1. For triangles on the same base AC and between the same parallels, their perpendicular heights to AC are equal.
  2. Using the area formula, equal base and equal height imply equal areas.

Detailed Proof

Step 1 – Common Height from Parallels

Let triangles ABC and ADC share base AC. Suppose the line through B and D is parallel to AC. The perpendicular distance from that line to AC is constant everywhere, so the altitudes from B and D to AC are equal; call this common height h.

Step 2 – Write the Areas

[△ABC] = \(\tfrac{1}{2}\)·AC·h and [△ADC] = \(\tfrac{1}{2}\)·AC·h, since both use the same base AC and the same height h.

Step 3 – Conclude Equality

Therefore, [△ABC] = [△ADC], which proves that triangles on the same base and between the same parallels have equal areas.
